# AssistDebugStick
デバッグ棒(DebugStick)の利便性を向上させるプラグイン

# Usage
デバッグ棒でプロパティを持つブロックを左クリックする事で,変更項目がプラグインで対応している
プロパティであればパーティクルで変更内容を可視化します。

# Commands
* /ads visualization <true/false> - プロパティを視覚化するかどうかを設定します。
